**************************************************************************************************
Capitulo 2: Introduccion al Aprendizaje Automatico (Machine Learning)
**************************************************************************************************
1. Conceptos Claves
- Modelo
- Entrada (Input: X)
- Salida (output: y)
- Etiquetas (Labels, Targets)
- Caracteristicas (Features)
- Muestras (Instancias)
- Dimensionalidad
- Algoritmo

2. Tipos de Aprendizaje

2.1. Aprendizaje Supervisado
- Modelos que pueden predecir etiquetas basándose en datos de entrenamiento etiquetados.
- Las 2 tareas mas comunes son:
  -) Clasificación: Modelos que predicen etiquetas como dos o más categorías discretas
  -) Regresión: Modelos que predicen etiquetas continuas

2.2. Aprendizaje No Supervisado
- Modelos que identifican la estructura en datos no etiquetados.
- Las 2 tareas mas comunes son:
  -) Agrupamiento: Modelos que detectan e identifican grupos distintos en los datos
  -) Reducción de Dimensionalidad: Modelos que detectan e identifican estructuras de menor 
     dimensión en datos de mayor dimensión.

3. Algoritmos de Aprendizaje Automatico

3.1. Algoritmos de Máquinas de Soporte Vectorial (SVM): Las SVM son técnicas de clasificación y regresión que encuentran el hiperplano óptimo que separa las clases en un espacio de alta dimensión.

3.2. Algoritmos de Regresión: Modelan la relación entre distintas variables (features) utilizando una medida de error que se intentará minimizar en un proceso iterativo para poder realizar predicciones “lo más acertadas posible”.
- Regresión Lineal: MCO, Ridge, Lasso, ElasticNet, Lars, LassoLars, OMP
- Regresión Logística: LogisticRegression

3.3. Algoritmos basados en Instancia: Modelos de Aprendizaje para problemas de decisión con instancias o ejemplos (muestras) de datos de entrenamiento que son importantes o requeridos por el modelo.
- Self-Organizing Map (SOM)
- k-Nearest Neighbor (KNN)

3.4. Algoritmos de Arbol de Decisión: Modelan la toma de Decisión basado en los valores actuales (reales) de los atributos que tienen nuestros datos.
- Arboles de Clasificación y Regresión (CART)
- Random Forests

3.5. Algoritmos Bayesianos: Usan el Teorema de Bayes de probabilidad para problemas de Clasificación y Regresión.
- Naive Bayes
- Gaussian Naive Bayes
- Multinomial Naive Bayes
- Bayesian Networks

3.6. Algoritmos de Clustering: Se usan para agrupar datos existentes de los que desconocemos sus características en común o queremos descubrirlas.
- K-Means
- K-Medians
- Hierarchical Clustering

3.7. Algoritmos de Reducción de la Dimensionalidad: Buscan explotar la estructura existente de manera no supervisada para simplificar los datos y reducirlos o comprimirlos.
- Principal Component Analysis (PCA)
- t-Distributed Stochastic Neighbor Embedding (t-SNE)

3.8. Algoritmos de Ensamble: Combinan las predicciones de varios modelos base para mejorar la precisión y la robustez de los resultados.
- Bagging: Random Forest
- Boosting: AdaBoost, Gradient Boosting (XGBoost)

3.9. Algoritmos de Redes Neuronales (ANN): Se usan para problemas de Clasificación y Regresión. Son muy buenas para detectar patrones.
- Perceptron
- Multi Layered Perceptron (MLP)
- Hopfield Networks
- Back-Propagation => Bolztman Machine (Hinton 1986)

3.10. Algoritmos de Aprendizaje Profundo (Deep Learning): Son la evolucion de las ANNs, con mas capas, datos, trabajo paralelo, etc.
- CNNs: LeNet, AlexNet, VGGNet, ResNet, RetinaNet, MobileNet, R-CNN, YOLO, etc.
- RNNs: RNN estandar, BRRNs, LSTMs, GRUs, Seq2Seq
- Transformers: ViT, DETR, SwT, CvT, DINAT, OneFormer, SAM, SAM2, etc.

3.11. Algoritmos de Aprendizaje por Refuerzo: Un agente aprende a tomar decisiones secuenciales. El agente recibe recompensas o castigos y tiene como objetivo maximizar la recompensa.
- Q-Learning
- Deep Q-Networks (DQN)

4. Diferencias Fundamentales entre ML Vs DL
- ML: Algoritmos muy rapidos (bajo procesam.) y se entrenan con regular data (10,000-100,000) =>   CPUs
- DL: Algoritmos mas lentos (mucho proces.) y se entrenan con bastante data (100,000-1'000,000) =>  GPUs
  Mucho Procesamiento => Aprender Millones de Pesos
  1. Entrada (Miles o Millones de Muestras)
  2. Muchas Capas Ocultas
- LMs: LLMs, LVMs, Miles de Millones de Pesos => Data Centers GPUs

5. Ejemplos de Tareas en Machine Learning
- Clasificacion Binaria: Encontrar si pertenece a una u otra categoria. Ej: Sexo (H-M), Lentes (Si,No), Bigotes (Si,No)
- Clasificacion MultiClase: Encontrar si pertenece a una categoria de muchas. Ej: Digito (0..9), Edad (1-100), Transito (+60)
- Regresion Simple: Predecir un Valor con una sola columna de entrada. Ej: Metros => Precio Casa, Azucar => Diabetes
- Regresion Multiple: Predecir un Valor con varias columnas de entrada. Ej: Metros, Zona, Cuartos, Pisos => Precio Casa
- Agrupamiento: Encontrar grupos de datos similares. Ej: Fotos de Personas => Grupo1 (Hombres), Grupo2 (Mujeres)
